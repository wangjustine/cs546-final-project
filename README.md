# Task Flow
TaskFlow is a collaborative task management web application that helps both individual and team projects streamline their workflow. The system will provide an interface for managing tasks and visualize progress with real-time updates like whiteboard sticky notes.

## Description/Core Features
Landing page: outlines purpose and functions of the application 
Main page:
Task management dashboard
After logging in, users are brought to a main board where existing progress will be displayed
Functions to create, edit, delete, mark tasks as complete
View task: expands with full details 
Task details include title, description, priority level, and completion status
Task categories/labeling
Deadline setting:
Users can set interim due dates for task deadlines
Overdue tasks will be highlighted for visibility

## Installation
How to set up the project locally:

```bash
git clone https://github.com/wangjustine/cs546-final-project.git
cd cs546-final-project
npm install
npm run seed
npm start
```

Then go to localhost:3000 to nagivate through our project.
